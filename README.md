#Habit Tracker
Habit Tracker built with HTML,CSS and Javascript.

##Features

-Add habits
-Set daily goal
-Track daily progress
-Delete habits
-LocalStorage support
-Enter key support
-Responsive design

## Technologies

-HTML5
-CSS3
-JavaScript (ES6)
